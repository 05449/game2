# EVADE∞（イベイド・インフィニティ）

> 「1秒でも長く生き残れ」

シンプルな操作で障害物を避け続ける、中毒性の高いアクションゲーム。

![EVADE∞](https://via.placeholder.com/800x400/0a0a1a/00ffff?text=EVADE%E2%88%9E)

## 🎮 遊び方

### ゲームの目的
画面上に出現する障害物を避けて、できるだけ長く生存してください。

### 操作方法

#### PC
| 入力 | アクション |
|------|-----------|
| マウス | プレイヤーがマウスを追従 |
| W/A/S/D または 矢印キー | 移動 |
| スペースキー | スローモーション（クールダウン20秒） |
| Shiftキー | 緊急回避（クールダウン30秒） |

#### スマートフォン
| 入力 | アクション |
|------|-----------|
| タッチ＆ドラッグ | プレイヤー移動 |
| ダブルタップ | スローモーション |

## 🎯 ゲームシステム

### 障害物の種類

| 名称 | 特徴 | 出現時間 |
|------|------|---------|
| ノーマル（赤） | 基本的な直線移動 | 0秒〜 |
| ファスト（オレンジ） | 小さくて高速 | 15秒〜 |
| ラージ（深紅） | 大きくて低速 | 20秒〜 |
| ホーミング（紫） | プレイヤーを追尾 | 30秒〜 |
| スプリッター（緑） | シールドで防ぐと分裂 | 45秒〜 |
| ウェーブ（水色） | 波状に移動 | 60秒〜 |
| ゴースト（青紫） | 点滅して見えにくい | 90秒〜 |
| レーザー（黄） | 超高速 | 120秒〜 |

### パワーアップ

| アイテム | 効果 |
|---------|------|
| 🛡️ シールド | 1回のみダメージ無効 |
| ⏱️ スロー | 5秒間ゲーム速度50% |
| 🔽 小型化 | 8秒間当たり判定50%縮小 |
| 🧲 マグネット | 6秒間近くの障害物を弾く |
| ⭐ タイムボーナス | スコアボーナス |

### スキル

| スキル | 効果 | クールダウン |
|--------|------|-------------|
| スローモーション | 3秒間ゲーム速度30% | 20秒 |
| 緊急回避 | 0.5秒間完全無敵 | 30秒 |

## 🏆 スコア

- **生存時間**がそのままスコアになります
- **ニアミス**でボーナス！ギリギリを狙え！
- ローカルにハイスコアが保存されます

## 🚀 起動方法

### 方法1: VS Code Live Server
1. VS Codeで「Live Server」拡張機能をインストール
2. `index.html`を右クリック → 「Open with Live Server」

### 方法2: ローカルサーバー
```bash
# Node.js
npx serve

# Python
python -m http.server 8080
```

### 方法3: 直接開く
`index.html`をブラウザにドラッグ＆ドロップ
（一部機能が制限される可能性があります）

## 🛠️ 技術仕様

- **言語**: HTML5 / CSS3 / JavaScript (ES6+)
- **描画**: Canvas API
- **音声**: Web Audio API（プログラム生成）
- **データ保存**: LocalStorage

## ✨ 特徴

- **外部ライブラリ不使用** - Pure JavaScript
- **音声ファイル不要** - Web Audio APIでリアルタイム生成
- **画像ファイル不要** - 全てCanvasで描画
- **レスポンシブ対応** - PC/スマートフォン両対応
